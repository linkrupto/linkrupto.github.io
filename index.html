How to share a secret s?

问题描述

怎样把s分成n份，使得任意不少于t份组合，能恢复出s？

缘起

《组合数学》（Chung Laung Liu，刘炯朗，台湾清华大学校长）中的一个数学问题：

Eleven scientists are working on a secret project. They wish to lock up the documents in a cabinet so that the cabinet can be opened if and only if six or more of the scientists are present. What is the smallest number of locks needed? What is the smallest number of keys to the locks each scientist must carry?

Problem Description

The whole set is U (|U|=m), divided into 11 subsets and each is of size n, s.t. only the union of any six or more sub-sests can recover the whole set U. Give m,n.

Solution

1)Any less than six set will not know at least one element in U.

lattice的几何结构

  1. L(A), A=一个m乘以n的=n个m维向量;
  2. L(A)就是这n个向量的整数系数线性组合；
  3. span(L(A))=span(A), 系数为任意实数的线性组合，L(A)扩张
  4. P(A): fundamental parallelepiped, span(A)的基本组成单元，格点间空的部分

lattice的基

  1. A是L(A)的基，除了A之外，还存在其它的基A’, 并且存在性质：

    A’= AZ ; |A’|=|A|

  2. lattice的基的“质量”好坏，对于解决lattice的问题有影响。通常要选一组“短的，正交”基。
    根据某一标准，选择基的过程，叫做lattice reduction。

Lattice不同质量的基，对解决困难问题影响的实例？

  1. lattice reduction存在很多不同的算法，比如： LLL，KZ， MInkowski，Seysen，etc. 不同算法也有不同应用。
    3.1 计算效率高，但是weak：LLL-polynomial in lattice dimension
    3.2 计算效率低，但是strong：HKZ(Kannan,Helfrich-Schnorr;Kumar-Sivakumar)
    3.3 between 3.1和3.2： Schnorr’s BKZ (2-BKZ=LLL, d-BKZ=HKZ)

  2. 一般解决CVP，SVP问题通过：sphere decoding方法

    STEP 1： lattice reduction，找质量好的格基和对偶格基
    STEP 2： 搜索，Schnorr-Euchner，within a hypersphere

    通常，第一步是为了降低第二步的效率

    4.1 successive minima

    一个下确界值r，第i-th successive minima表示，格与半径为r的“球体”相交，交集中的格点的扩张至少为i维空间。

  3. KZ在并行CVP问题（即，同一个格基，不同的指定向量）中，对于STEP 2的效率提高更显著，即使KZ本身的效率并不如LLL好。

Hash Function

一类将任意长度输入压缩成更短输出的函数。经典应用包括，数据结构中的hash table，可实现$\bigO(1)$复杂度查找。（collison越少，效率越高）

hash function是random function吗？

hash function in cryptography

  1. one-wayness
  2. collision resistance

Keyed hash function

  1. $\textbf{Gen}(1^\lambda)\rightarrow s$: 随机算法，产生密钥s。
  2. $\textbf{H}(s,m)\rightarrow H^s(m)$: 确定性算法，$m\in\{0,1\}^$,即输入可为任意长度， 输出$H^s(m)\in\{0,1\}^{\ell(n)}$。

最近家里东西堆积过多，准备清理一下，试着在网上二手市场叫卖一下，想着也能创手一笔。想起大学的时候，临近毕业了，楼道的公告栏里总会贴出许多宿舍的售卖清单，花花绿绿，粗粗细细的字体，甚是可爱，多是一些衣服、包、杂志书籍之类的，大约是受学校附近夜市的感染和启发。清理之后，多出的书卖给了多抓鱼，寄运方便，根据品相打款，看在快递的份上贱卖就贱卖了吧，省了不少事也省了一笔开销。另外一些杂物，包、首饰、耳机、明星周边等等挂到了闲鱼。
