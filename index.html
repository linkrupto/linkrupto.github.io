How to share a secret s?

问题描述

怎样把s分成n份，使得任意不少于t份组合，能恢复出s？

缘起

《组合数学》（Chung Laung Liu，刘炯朗，台湾清华大学校长）中的一个数学问题：

Eleven scientists are working on a secret project. They wish to lock up the documents in a cabinet so that the cabinet can be opened if and only if six or more of the scientists are present. What is the smallest number of locks needed? What is the smallest number of keys to the locks each scientist must carry?

Problem Description

The whole set is U (|U|=m), divided into 11 subsets and each is of size n, s.t. only the union of any six or more sub-sests can recover the whole set U. Give m,n.

Solution

WTO，银行与房地产

Veröffentlicht am 2019-04-02 Bearbeitet am 2019-06-30 Kommentare:

读到耐大十几年前博文，提到外资银行入华的事情。根据中国入世的承诺，2006年12月是外资银行进入中国（即中国银行业对外开放）的最后期限。她认为，外资银行入华是“轻装上市”，因没有国有银行那些因房市泡沫产生的恶性放贷包袱，也无需因地方政府行政干预而放贷救市，所以理应更具优势，易吸引储户搬家，自主选择优良资产放贷，形成良性循环。

原来国内的房地产泡沫并不是近些年才产生的，而是自2000年后，从一线城市开始蔓延开，房价持续高速上涨，房地产业建设过度，所以现在能看到很多“空城”。耐大也指出，房地产铁三角（政府、开发商、银行）谁都不希望房价回落甚至楼市泡沫破灭，否则开发商破产，银行放给开发商的贷款只剩下一堆钢筋水泥，坏账增加。

13年之后的今天看来，外资银行并没有太多出现在百姓的日常生活中，更没有出现外资银行进入而导致国有银行入不敷出没钱放贷的情况，并且四大国有银行因为盈利高到超过所有其它行业而上了微博热搜（2018年六大行净盈利1.06万亿，换算成日赚，工商银行日均净盈利8.16亿，建设银行日均净盈利近7亿，农业银行日均净盈利为5.6亿，中国银行日均净盈利近5亿，交通银行日均净盈利2亿，邮储银行日均净盈利1.43亿）,房地产市场泡沫还未破裂，交易量时增时减，伴随着房产税时不时要出台的消息。

世贸组织（WTO）是个什么组织？

124个国家在1994年4月15日签订的马拉喀什协定（Marrakesh Agreement）中提到于1995年1月1日成立WTO，用于规范国际贸易。WTO也是原关税与贸易总协定（GATT，成立于1948年）的替代组织。

GATT成立于1947年，1948年开始生效，由23个国家签订（包括中华民国），1998年发展为128个国家，并改为WTO。GATT/WTO至今共历经9轮磋商谈判，最后一轮始于2001年的多哈回合贸易谈判未能达成协议，2008年宣布破局。

GATT/WTO组织最重要的两大的作用是监管协议的实施、管理和运作以及提供论坛作为贸易磋商和解决贸易争端的场所。
