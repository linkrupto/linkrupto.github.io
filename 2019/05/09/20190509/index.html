Hash Function

一类将任意长度输入压缩成更短输出的函数。经典应用包括，数据结构中的hash table，可实现$\bigO(1)$复杂度查找。（collison越少，效率越高）

Keyed hash function

  1. $\textbf{Gen}(1^\lambda)\rightarrow s$: 随机算法，产生密钥s。
  2. $\textbf{H}(s,m)\rightarrow H^s(m)$: 确定性算法，$m\in\{0,1\}^$,即输入可为任意长度， 输出$H^s(m)\in\{0,1\}^{\ell(n)}$。
